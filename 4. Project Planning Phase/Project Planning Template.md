# Project Planning Template

## Project Scope
- **In-scope:** 
  - Importing and cleaning the `Product Positioning.csv` dataset.
  - Creating 8 specific visualizations utilizing key Calculation Fields (Price, Competitor Price, Sales Volume).
  - Building an interactive dashboard with utilization of filters.
  - Creating a 3-scene Tableau Storyboard.
  - Publishing the workbook to Tableau Public.
  - Building a Flask web application to embed the Dashboard and Story.
