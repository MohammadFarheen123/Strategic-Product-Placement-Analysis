# Product Placement Analysis Dashboard

This project analyzes the impact of product positioning, promotions, and consumer demographics on sales volume in an FMCG retail context.

## Tableau Public Dashboard
We have recreated this web-based dashboard in Tableau to provide dynamic, interactive data visualization. 

**View the Live Interactive Dashboard Here:** 
👉 https://public.tableau.com/views/PRODUCTPLACEMENTANALYSISDASHBOARD_17830883915970/Dashboard1?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

### Key Insights Analyzed:
- Average Sales Volume vs. Product Category
- Seasonal Sales Trends by Category
- Consumer Demographics Breakdown
- Competitor Price vs. Our Price Correlation
- The Impact of Promotions on Sales Volume
- Foot Traffic Relationship with Sales
- Cross-tabulation of Product Category and Placement Position

## Setup
The dataset `Product Positioning.csv` contains all the raw data used for the visualization.
