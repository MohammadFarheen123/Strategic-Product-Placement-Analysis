# Technology Stack

## Frontend / Visualization
* **Tableau Desktop / Tableau Public** - Reason for choice: Industry-standard for creating interactive dashboards and storyboards. Publishing to Tableau Public allows for free, accessible web embedding.
* **HTML/CSS (Web UI)** - Reason for choice: Provides the structural wrapper and styling for embedding the Tableau Public `<iframe>` or JS embed script.

## Backend / Web Server
* **Python (Flask)** - Reason for choice: A lightweight micro web framework perfectly suited for serving a simple web UI that hosts embedded visualizations securely and quickly.

## Data Source / Storage
* **CSV (Comma Separated Values)** - Reason for choice: A lightweight, universal format perfectly suited for the Kaggle dataset.
* **Kaggle** - Reason for choice: Reliable source for the finalized "Impact of Product Positioning on Sales" dataset.
