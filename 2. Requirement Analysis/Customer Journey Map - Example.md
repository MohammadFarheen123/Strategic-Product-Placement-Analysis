# Customer Journey Map

## Persona: Retail Store Manager
## Scenario: Analyzing product placement performance to optimize store layout for the upcoming season.

| Stages | 1. Data Collection | 2. Dashboard Exploration | 3. Insight Generation | 4. Action & Optimization |
|--------|--------------------|--------------------------|-----------------------|--------------------------|
| **User Goals** | To access reliable, clean sales data | To view key metrics at a glance | To understand *why* certain products sell well | To change store layout to maximize sales |
| **Actions** | Downloads the finalized dataset from Kaggle. | Opens the Tableau Dashboard and checks the "Avg Sales Volume vs Product Category" chart. | Uses interactive filters to isolate "Clothing" in "Front of Store" during "Seasonal" spikes. | Moves clothing displays to the front of the store based on the data. |
| **Touchpoints**| Kaggle, Data Source Tab in Tableau | Tableau Dashboard UI, Filter Checkboxes | 3-Scene Tableau Storyboard | Store physical layout planogram |
| **Emotions** | Anxious but hopeful | Impressed by the clean, responsive design | Confident and informed | Empowered |
| **Pain Points**| Ensuring no null values or formatting errors. | Too much data can be overwhelming without good design. | Explaining the insights to upper management without visual aids. | Executing physical changes takes time. |
| **Opportunities**| Provide a pre-cleaned, validated dataset. | Implement clear color coding. | Use the 3-scene storyboard to seamlessly communicate findings. | Track ROI of the layout changes in the next quarter's data. |
